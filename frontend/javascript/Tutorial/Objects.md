# Objects

## Declaration

Objects are declared using {}

```javascript
let object = {
    attribute1="value1",
    attribute2=value2,
    attribute3=value3,
    ...
}
```

## Adding an attribute

```javascript
object.newAttribute = "value"
```

## Access the value of an attribute

The value of an attribute is accessed using "." .

```javascript
someVariable = object.attribute
```
